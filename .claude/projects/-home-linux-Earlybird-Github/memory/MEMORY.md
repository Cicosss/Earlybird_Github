# EarlyBird Memory

## Modalità COVE (Chain of Verification)

COVE è attiva in questo progetto. Può essere invocata con `/cove`.

### Struttura
- **Skill file**: `.claude/skills/cove.md`
- **Documentazione**: `CLAUDE.md` (sezione COVE Mode)

### Protocollo
1. FASE 1: Bozza preliminare (ipotesi)
2. FASE 2: Verifica avversariale (domande critiche)
3. FASE 3: Verifiche indipendenti (solo conoscenza pre-addestrata)
4. FASE 4: Risposta definitiva (correzioni documentate)

### Quando attivare COVE
- Codice critico che richiede accuratezza
- Verifica di API, sintassi, parametri
- Fact-checking di date, versioni, numeri
- Debugging complesso
